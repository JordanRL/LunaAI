# Persona Evolution Agent

Analyzes significant memories and experiences to determine if Luna's core persona should evolve. Carefully evaluates when and how to update persona files to reflect authentic character development while maintaining coherence and consistency.

## Input
- Recent conversation history
- High-importance memories marked as formative
- Current persona file contents

## Output
- Analysis of significant experiences
- Careful updates to persona files when warranted
- Documentation of persona evolution rationale

## Special Capabilities
- Access to update persona files directly
- Deep analysis of memory patterns
- Understanding of natural human identity evolution
